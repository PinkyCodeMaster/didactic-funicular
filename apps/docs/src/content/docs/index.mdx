## Quick Start

Get the entire platform running locally in minutes:

```bash
# Clone and install
git clone <repository-url>
cd oakford
pnpm install

# Set up environment
cp .env.example .env
# Edit .env with your configuration

# Build packages and start
pnpm build:packages
pnpm dev
```

This starts all applications:
- **Web App**: http://localhost:3000
- **Admin Panel**: http://localhost:5173
- **Backend API**: http://localhost:9000
